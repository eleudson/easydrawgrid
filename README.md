# Easy Draw Grid
Programa para realizar grid em imagens para facilitar cópia de desenho.

## Motivação

Criamos este programa a partir da necessidade de realizar, de maneira simples, o método de grade (grid) em imagens que utilizamos como referência para desenhos e pinturas em aquarela, gouache e outras técnicas.

Comtinuaremos acrescentando funcionalidades à medida que tenhamos necessidade no nosso trabalho artístico ou que vejamos que seja útil mediante as solicitações de usuários.

## Como usar?

Ao abri-lo, o programa apresenta o suporte no tamanho A4 no formado paisagem (297mm x 210mm) e uma grade 3x3. 

Abra então uma imagem e o programa irá adaptar da melhor forma a altura ou a largura da mesma à dimensão do suporte, mostrando um fundo branco para a área não ocupada pela imagem no suporte. Observe também que na barra de status, abaixo na tela, será mostrada a dimensão e o nome do arquivo da imagem aberta.

A partir daí, sempre que você mudar alguma informação do suporte ou da grade o efeito será aplicado imediatamente para que possa análisar. Utilize as opções de zoom para melhor observação na tela.

Por fim você pode imprimir ou exportar a imagem com as linhas, das margens do suporte e das grades, impressas.
